# Поминалка — Бот-напоминалка

## Демо
- /start → главное меню
- Календарь, редактирование, уведомления
## Функции
- Календарь с "сегодня"
- Время: 1830, 18 30
- Редактирование и отмена
- Уведомления 24/7

## Запуск
1. Создай .env: BOT_TOKEN=твой_токен
2. pip install -r requirements.txt
3. python bot.py
